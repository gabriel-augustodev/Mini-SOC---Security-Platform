# 🛡️ Mini SOC - Security Platform

Plataforma de monitoramento e análise de segurança (Mini SOC) desenvolvida para simular um ambiente real de detecção de vulnerabilidades e análise de risco.

## 🚀 Funcionalidades

- 🔍 Scan de vulnerabilidades utilizando Nmap
- ⚙️ Integração Node.js + Python
- 📊 Análise automática de risco
- 📦 API REST para execução de scans

## 🧠 Arquitetura

Cliente → Node.js → Python → Nmap → Node.js → Cliente

## 🛠️ Tecnologias utilizadas

- Node.js
- Python
- Nmap
- Express
